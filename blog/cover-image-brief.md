# Cover Image Brief

## Concept
A clean, minimal architecture diagram showing the key trust boundary: developer laptop → private network → internal ALB → ECS → Bedrock. The visual hook is the "zero credentials on the laptop" idea — the laptop side has a person/cursor icon and a `claude /login` browser window; the AWS cloud side has the lock/key icon.

## Style
- Dark background (#0f1117 or similar deep navy/charcoal)
- AWS-orange (#FF9900) for the AWS cloud box
- White or light-grey for text labels
- Minimal — no decorative icons beyond the AWS logo and a simple laptop silhouette
- 1000×420px (dev.to cover image recommended ratio)

## Text overlay (optional)
- Large: **"Claude Code for Teams"**
- Small: **"Self-hosted · Zero API keys on developer machines · AWS-native"**

## Tools that can generate this
- Excalidraw (hand-drawn style, exports PNG)
- Figma (precise layout)
- AWS Architecture Diagram tool
- DALL-E / Midjourney prompt: *"minimal dark-background cloud architecture diagram, developer laptop connecting over VPN to internal AWS load balancer, zero credentials text label, AWS orange accent colour, clean tech illustration"*
